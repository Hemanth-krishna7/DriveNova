# Layout Rules

## General Rules

- Never build perfectly centered compositions.
- Never use equal-width two-column layouts.
- Never rely on Tailwind's default spacing aesthetics.
- Whitespace is a design element.
- Avoid visual symmetry.

---

## Editorial Composition

- Typography dominates the composition.
- Vehicle supports the composition.
- Composition always comes before decoration.
- The eye should naturally move through the scene rather than immediately focusing on the car.

---

## Navigation

Navigation should always feel:
- floating
- detached from the page
- premium
- lightly transparent
- spacious

*Navigation should never resemble a default website header.*

---

## Hero Philosophy

1. Editorial first.
2. Atmosphere second.
3. Typography third.
4. Vehicle fourth.

*The Ferrari enhances the composition. It never dominates the composition.*

---

## Section Philosophy

Each section should feel like a cinematic scene rather than a container.

### Avoid
- obvious boxes
- repetitive grids
- dashboard layouts
- identical spacing patterns

---

## Depth Rules

Every major section should contain multiple visual layers.

### Minimum Layers
- Background
- Midground
- Foreground

*Motion should reinforce these layers. Depth should come from composition rather than excessive animation.*

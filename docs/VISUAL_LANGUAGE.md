# Visual Language

## Color Philosophy

### Background
- Very dark charcoal / near-black

### Accent
- Electric blue

### Secondary Accent
- Chrome / silver

### Typography
- White
- Soft gray hierarchy

### Avoid
- Bright gradients as decoration
- Rainbow accents
- Oversaturated colors

---

## Glass Philosophy

Glass should always be:
- subtle
- premium
- lightly blurred
- thin bordered
- physically believable
- never overly transparent

---

## Spacing Philosophy

Use generous whitespace.

Allow elements to breathe.

Avoid crowded layouts.

Avoid perfectly equal spacing.

Asymmetry is preferred.

---

## Card Philosophy

Cards should feel like floating architectural objects.

### Avoid
- boxed UI
- dashboard aesthetics
- repetitive card grids
